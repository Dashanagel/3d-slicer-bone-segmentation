# 3D CT Bone Segmentation Using 3D Slicer

Author: Dashana Gelin

## Project Overview

This project demonstrates basic medical image processing using 3D Slicer. The objective was to visualize CT scan data, perform bone segmentation, and generate a 3D anatomical model.

3D Slicer is widely used in medical imaging research for visualization, segmentation, and 3D reconstruction of medical datasets.

## Tools Used

- 3D Slicer
- CT scan dataset
- DICOM imaging format

## Workflow

### 1. Import CT Scan

A CT scan dataset was loaded into 3D Slicer and explored in three orthogonal views:

- Axial
- Sagittal
- Coronal

### 2. Bone Segmentation

The Segment Editor module was used to isolate bone structures.

Tools used:
- Threshold
- Paint
- Smoothing

### 3. 3D Model Generation

After segmentation, the bone structures were rendered into a 3D model.

## Results

The workflow successfully produced:

- CT slice visualization
- Bone segmentation mask
- 3D anatomical reconstruction
